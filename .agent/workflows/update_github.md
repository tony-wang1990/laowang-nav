---
description: 如何将代码更新推送到 GitHub
---

当您修改了代码并希望同步到 GitHub 仓库时，请按照以下步骤操作：

1.  **打开终端** (Ctrl+` 或在 VS Code 菜单中选择 终端 -> 新建终端)

2.  **添加更改**
    将所有修改过的文件添加到暂存区：
    ```bash
    git add .
    ```

3.  **提交更改**
    填写本次更新的说明（例如："修复了登录页面的 bug"）：
    ```bash
    git commit -m "在此处填写您的更新说明"
    ```

4.  **推送到 GitHub**
    将本地提交推送到远程仓库：
    ```bash
    git push
    ```

🎉 完成！您的代码现在已经更新到 GitHub 上了。
